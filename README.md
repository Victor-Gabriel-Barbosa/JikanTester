# JikanTester

Aplicação Web para testar a API Jikan

## Descrição

Este repositório contém uma aplicação web para testar a API do Jikan, que é uma API pública não oficial do MyAnimeList. A ideia é facilitar a interação com a API, permitindo que usuários realizem solicitações e vejam as respostas de forma intuitiva e amigável.

**🌐 Acesse a plataforma:** [github.io/JikanTester](https://victor-gabriel-barbosa.github.io/JikanTester/)

## Funcionalidades

- **Testar Endpoints**: Execute chamadas para diversos endpoints da API do Jikan.
- **Visualizar Respostas**: Veja as respostas no formato JSON diretamente na interface da aplicação.
- **Exibir Erros**: Se uma chamada falhar, a aplicação mostrará uma mensagem de erro clara.
- **Interface Intuitiva**: Desenvolvida para proporcionar uma experiência de usuário amigável e fluida.

## Tecnologias Utilizadas

- **HTML**: 28.8% - Estrutura e semântica da aplicação
- **CSS**: 20.7% - Estilização e layout responsivo
- **JavaScript**: 50.5% - Lógica da aplicação e integração com a API Jikan

## Como Executar o Projeto

1. Clone o repositório:
   ```bash
   git clone https://github.com/Victor-Gabriel-Barbosa/JikanTester.git
   ```

2. Navegue até o diretório do projeto:
   ```bash
   cd JikanTester
   ```

3. Abra o arquivo `index.html` em um navegador web de sua preferência.

## Sobre a API Jikan

A API Jikan é uma API REST não oficial do MyAnimeList que fornece dados sobre:
- Anime
- Manga
- Personagens
- Pessoas
- E muito mais!

Para mais informações, visite: [https://jikan.moe/](https://jikan.moe/)

## Como Usar

1. Abra a aplicação no navegador
2. Selecione o endpoint que deseja testar
3. Insira os parâmetros necessários
4. Clique em "Enviar" ou "Testar"
5. Visualize a resposta da API em tempo real
